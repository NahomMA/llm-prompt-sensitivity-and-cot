\noindent This project investigates two fundamental areas of digital image processing: geometric transformation and image enhancement. In the first task, we applied a composite affine transformation involving scaling, shearing, rotation, and translation to the reference image \texttt{lena.png} to create a distorted version. We then registered the distorted image back to the original using inverse mapping with bicubic interpolation. The registration accuracy was validated through multiple similarity metrics, achieving a Normalized Cross-Correlation of 0.965 and Normalized Mutual Information of 1.477, both indicating excellent alignment. The second task focused on enhancing the visual quality of \texttt{clown.png} using various point processing techniques. We implemented contrast stretching, histogram equalization, and gamma correction to improve image contrast and brightness. The effectiveness of each enhancement method was assessed by comparing the processed images and analyzing their intensity histograms. Results demonstrate the practical utility of these techniques for improving image quality and achieving accurate geometric registration.
